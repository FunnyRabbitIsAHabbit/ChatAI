# Groq console AI assistant

## Read before anything else

### Clone repository
    `git clone https://github.com/FunnyRabbitIsAHabbit/ChatAI.git`

### Go into folder
    `cd ChatAI'

### Create your own 'config.py'
    `nano config.py'

And set your GROQ API key

*Example:*
    `GROQ_API_KEY="YOUR_API_KEY"`

### Run python app
    `python3.12 -m main'
